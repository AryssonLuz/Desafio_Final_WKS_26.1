# Projeto Completo de Dados: Modelagem, SQL, Python e Power BI

## Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio prático com o objetivo de avaliar conhecimentos técnicos em diferentes etapas do fluxo de dados.

O desafio envolveu desde a modelagem de banco de dados até a análise e visualização de dados, passando por SQL, Python (ETL e Machine Learning) e Power BI.

Cada etapa foi desenvolvida de forma individual, mas integrada, simulando um cenário real de projeto de dados.



##  Objetivo

Demonstrar habilidades práticas em:

* Modelagem de dados (conceitual e lógica)
* Manipulação de banco de dados com SQL
* Processos de ETL e análise com Python
* Aplicação de Machine Learning
* Criação de dashboards interativos no Power BI
* Documentação técnica



##  Etapas do Projeto

### 1. Modelagem de Dados

Foi realizada a modelagem de uma empresa fictícia de aluguel de carros.

* Criação do modelo conceitual
* Criação do modelo lógico
* Definição de entidades, atributos e relacionamentos
* Estruturação de tabelas relacionais



### 2. SQL

A partir do modelo lógico, foi implementado o banco de dados utilizando SQL.

* **DDL (Data Definition Language):**

  * Criação do banco de dados e tabelas

* **DML (Data Manipulation Language):**

  * Inserção de dados nas tabelas

* **DQL (Data Query Language):**

  * Consultas para análise de dados
  * Interpretação dos resultados

Documentação detalhada disponível na pasta `docs`.



### 3. Python (ETL e Machine Learning)

Nesta etapa, foi utilizado o dataset do **Banco Box**.

#### ETL

* Tratamento de dados
* Limpeza e preparação
* Ajustes para análise

#### Análise de Dados

* Exploração inicial dos dados
* Visualizações básicas

#### Machine Learning

Foram utilizados quatro modelos de classificação para prever risco de crédito:

* Random Forest
* Decision Tree
* Logistic Regression
* KNN (K-Nearest Neighbors)

O objetivo foi comparar os modelos e identificar o mais adequado para previsões futuras.



### 4. Power BI

Foi utilizado o dataset já tratado na etapa de Python.

#### Dashboard desenvolvido com:

* Cartão: Média de renda
* Cartão: Total de clientes
* Gráfico de pizza: Distribuição de risco de crédito
* Gráfico de linhas: Renda por idade
* Gráfico combinado (barra + linha): Profissões vs endividamento e risco

#### Principais insights:

* Distribuição equilibrada entre os níveis de risco
* Leve predominância de risco alto
* Renda média de aproximadamente R$ 10.020
* Profissões com maior nível de endividamento
* Relação entre idade e estabilidade de renda

 As interpretações completas estão documentadas na pasta `docs`.



## Tecnologias Utilizadas

* Python (Pandas, Scikit-learn, Matplotlib)
* Power BI
* Modelagem de Dados (brModelo ou similar)
